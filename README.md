# phaser-game


```shell
 git clone git@github.com:WilliamHenckel/phaser-game.git
```
```shell
 cd phaser-game
```

## Installation
```shell
yarn install
```

## Run
```shell
yarn run
```
### Run on specific port
```shell
yarn run -- -p 666
```
<!-- option for http-server : -->
<!-- -p Port to use (defaults to 8080) -->

now open http://127.0.0.1:7070
